## 📘 Amazon Demo Scraping Tool（デモ版）

Python（Requests / BeautifulSoup / Pandas）を使用して、  
**Amazon風のデモECサイト** から商品情報を抽出するツールです。

---

### 🔍 取得項目
- 商品タイトル  
- 価格  
- 在庫（デモサイトのため **N/A**）  
- 詳細ページURL  

---

### 🌐 使用サイト（Amazon風デモ）
https://webscraper.io/test-sites/e-commerce/static/computers/laptops

---

### ⚙️ 機能
- HTML解析（Requests + BeautifulSoup）によるデータ抽出  
- 商品リスト **20件** を CSV に保存  
- スクレイピング案件の **テスト・練習用として最適**

---

### 📄 出力ファイル
- **amazon_demo_products.csv**

---

### 📂 ソースコード
- `amazon_demo_scraper.py`
